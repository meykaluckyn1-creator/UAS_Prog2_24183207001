# penjelasan singkat
Konsep OOP yang Digunakan

Abstract Class (User)
Digunakan sebagai class induk untuk Mahasiswa dan Dosen, serta memiliki method abstract tampilInfo().

Inheritance & Hierarchical Inheritance
Class Mahasiswa dan Dosen mewarisi User dan meng-override method tampilInfo() sesuai kebutuhan masing-masing.

Encapsulation
Semua atribut dibuat private dan diakses menggunakan constructor, getter, dan setter.

Polymorphism
Object Mahasiswa dan Dosen diperlakukan sebagai User saat menampilkan data.

Interface (Penilaian)
Digunakan untuk perhitungan nilai akhir, dan diimplementasikan oleh class Nilai.
